# Sistema de Asistencia

Base de datos MySQL para un sistema de asistencia con marcado por CI.

## Contenido

- `sistema_asistencia.sql`: script principal de creacion de tablas, datos iniciales y consultas de referencia.

## Uso

1. Crear la base de datos `sistema_asistencia` en MySQL.
2. Importar el archivo `sistema_asistencia.sql`.
3. Verificar las tablas y los datos semilla cargados.

## Requisitos

- MySQL 8.0 o superior.
